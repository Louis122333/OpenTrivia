Instructions

1. Select the number of questions, difficulty, and category.

2. Answer each question within the time limit.

3. Your score will be displayed at the end of the quiz.

About the API

This quiz app uses the Open Trivia Database API. The Open Trivia Database is a free-to-use, user-contributed trivia question database. It provides a wide range of trivia questions from various categories and difficulties.

For more information about the API, visit the Open Trivia API Documentation.
